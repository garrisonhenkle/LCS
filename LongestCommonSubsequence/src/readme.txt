g++ -std=c++11 -o main *.cpp

Commands
-----------
Compilation: g++ -std=c++11 -o main *.cpp

-	Compiles the program to the script main

Running: ./main arg1 arg2

-	arg1 and arg2 should both be strings

-	The output will be the longest common subsequence on the first line and the 
	length of that subsequence on the second line

- 	Examples: ./main BDCABA ABCBDAB
-		  ./main vintner writers
